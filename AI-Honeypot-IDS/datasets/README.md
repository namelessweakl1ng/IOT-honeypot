# Datasets

This folder holds the canonical labeled dataset exported by the ML pipeline.

## File
- `honeypot_dataset.csv` — generated by `Computer-2-Analysis/ml/src/01_ingest.py`
  after running the attack suite.

## Schema
See `docs/09_Dataset.md` for the full schema and `docs/09_Dataset.md` §9.4
for the typical class distribution.

## How to regenerate
```bash
cd Computer-2-Analysis
source ~/.venvs/honeypot-ml/bin/activate
python ml/src/01_ingest.py
# → writes ../datasets/honeypot_dataset.csv
```

## Loading in third-party tools
```python
import pandas as pd
df = pd.read_csv("datasets/honeypot_dataset.csv")
print(df.shape)
print(df["attack_class"].value_counts())
```

The CSV is suitable for independent re-training in any ML framework
(scikit-learn, TensorFlow, PyTorch, Weka, RapidMiner, KNIME).

## Privacy
All `src_ip` values are private lab IPs (`192.168.1.30`, `172.20.0.1`).
No PII. Safe to share with evaluators and to include in the project report.
