# Global configs

Cross-project configuration files. Per-computer configs live under
`Computer-1-Honeypot/configs/` and `Computer-2-Analysis/configs/`.

## Files
- `hosts` — example `/etc/hosts` entries for all three machines
- `router-setup.md` — step-by-step router configuration guide
- `firewall-comp1.sh` — ufw rules for Computer 1
- `firewall-comp2.sh` — ufw rules for Computer 2
- `sshd-hardening.conf` — optional sshd_config hardening for all three machines
