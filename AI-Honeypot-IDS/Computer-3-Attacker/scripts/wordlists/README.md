# scripts/ wordlists

These wordlists are intentionally small so the demo runs quickly. Replace
with larger lists (rockyou.txt, SecLists) for more realistic brute force.

- `users.txt` — 20 common usernames
- `passwords.txt` — 27 common passwords (includes the honeypot's seeded
  creds: `admin`, `password`, `1234`, `toor`, `s3cr3t`, `alicepw`, `bobpw`)
- `dirs.txt` — ~60 paths and payloads specific to the honeypot's endpoints
