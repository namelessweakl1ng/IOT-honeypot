# Reports folder

Generated by `03_train.py`, `04_evaluate.py`, and `05_dataset_stats.py`:

- `cv_metrics.json` — cross-validation metrics for all 6 models
- `best_model.txt` — name of the best model (by test F1 macro)
- `summary.json` — full evaluation summary (accuracy, precision, recall, F1,
  confusion matrix, per-class report) for every model
- `model_comparison.csv` — friendly comparison table
- `confusion_matrix_<model>.png` — one per model
- `roc_curve_<best>.png` — OvR micro-averaged ROC for the best model
- `classification_report_<model>.json` — per-class precision/recall/F1
- `dataset_stats.md` — EDA summary
